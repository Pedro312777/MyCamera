# 📷 MyCamera

O **MyCamera** é um aplicativo mobile desenvolvido com **React Native** e **Expo** que utiliza a câmera do dispositivo para capturar fotos de forma simples e intuitiva. O projeto foi criado com o objetivo de praticar o uso da API de câmera do Expo, gerenciamento de permissões e desenvolvimento de interfaces para dispositivos móveis.

## 🚀 Funcionalidades

- 📸 Captura de fotos utilizando a câmera do dispositivo.
- 🔄 Alternância entre câmera frontal e traseira.
- 🔒 Solicitação de permissão para acessar a câmera.
- 📱 Interface simples e responsiva.

## 🛠️ Tecnologias Utilizadas

- React Native
- Expo
- Expo Camera
- TypeScript
- React Hooks (`useState`, `useEffect`, `useRef`)

## ▶️ Como executar o projeto

### 1. Clone o repositório

```bash
git clone https://github.com/Pedro312777/MyCamera,git
```

### 2. Acesse a pasta do projeto

```bash
cd MyCamera
```

### 3. Instale as dependências

```bash
npm install
```

ou

```bash
npx expo install
```

### 4. Execute o projeto

```bash
npx expo start
```

Depois disso, basta abrir o aplicativo pelo **Expo Go** ou em um emulador Android/iOS.
